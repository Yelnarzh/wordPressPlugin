OnePay Payment Gateway

Complete payment solution for WordPress & WooCommerce

💳 Payment Processing

Accept one-time payments via credit/debit cards through OnePay API integration

🔄 Subscriptions

Manage recurring payments with automatic billing and subscription lifecycle

💰 Refund Management

Process full and partial refunds directly from WordPress admin panel

🛒 WooCommerce Integration

Seamless checkout integration with WooCommerce for e-commerce stores

📋 Custom Forms

Add payment forms anywhere using simple shortcodes

📊 Transaction Tracking

Complete logging and status tracking in WordPress database

🚀 Quick Installation

  1. Zip the onepay-payment-gateway folder
  2. In WordPress admin, go to Plugins → Add New → Upload Plugin
  3. Upload the ZIP file and click Install Now
  4. Activate the plugin
  5. Navigate to OnePay menu and configure your API credentials
  6. Enable the gateway in WooCommerce → Settings → Payments
